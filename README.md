## 📖 Overview

The ATM Machine Simulation is a Java-based console application that mimics the basic functionalities of a real ATM. It allows users to securely access their account using a PIN and perform operations such as checking balance, depositing money, and withdrawing funds.

This project is built for educational purposes to strengthen core Java concepts and logic building.

---

## ✨ Features

- 🔐 PIN Authentication System  
- 💰 Balance Inquiry  
- 💵 Deposit Money  
- 💸 Withdraw Money  
- 🔁 Interactive Menu (Loop-based system)  
- ❌ Exit Option  

---

## ⚙️ How It Works

1. User enters PIN  
2. System validates PIN  
3. If correct → ATM menu is displayed  
4. User selects operation  
5. Balance updates accordingly  
6. Loop continues until exit  

---

## ▶️ How to Run

1. Clone the repository
```bash
git clone https://github.com/your-username/atm-machine.git

```

2. Navigate to the project folder
```bash
cd atm-machine
```

3. Compile the Java file
```bash
javac ATM_Machine.java
```

4. Run the program
```bash
java ATM_Machine
```
